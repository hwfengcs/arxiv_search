# arXiv 论文查找工具

这是一个Web应用程序，用于从arXiv上根据用户指定的关键词和时间范围（默认为最近7天）查找相关论文，并生成一个Markdown格式的报告。

## 功能特性

- **自定义论文搜索**: 用户可以输入任意关键词或完整的arXiv查询语句来搜索论文。
- **灵活的时间范围**: 用户可以指定查找最近N天内的论文，默认为7天。
- **现代化前端界面**: 重新设计了前端界面，提供更友好的用户体验。
- **动态论文展示**: 获取到的论文会以卡片形式清晰地展示在页面上。
- **Markdown报告生成**: 可以将当前查找到的论文信息整理成Markdown格式的报告并提供下载。

## 技术栈

- **后端**: Flask (Python)
- **前端**: HTML, CSS, JavaScript
- **数据获取**: `arxiv` Python库

## 如何运行

### 1. 环境准备

- 安装 Python 3.x
- 安装 pip (Python包管理器)

### 2. 安装依赖

进入 `backend` 目录，安装所需的Python包：

```bash
cd backend
pip install Flask flask-cors arxiv
```

### 3. 运行后端服务

在 `backend` 目录下运行Flask应用：

```bash
python app.py
```

后端服务默认会在 `http://127.0.0.1:5000` 上运行 (或者根据 `app.py` 中的配置，可能是 `http://0.0.0.0:5000`，允许局域网访问)。

### 4. 访问前端页面

直接在浏览器中打开 `frontend/index.html` 文件。

- **输入搜索主题**: 在 “搜索主题” 输入框中输入您感兴趣的关键词（例如 “language model”）或完整的arXiv查询语句（例如 `ti:"reinforcement learning" AND cat:cs.LG`）。如果只输入关键词，系统会默认在论文标题中搜索。
- **输入时间范围**: 在 “最近几天?” 输入框中输入您希望查找多少天内的论文（例如输入 `7` 代表查找最近7天的论文，这是默认值）。
- 点击 “查找论文” 按钮，应用会从arXiv获取相关论文并以卡片形式显示在页面上。
- 如果找到了论文，会出现 “下载报告” 按钮，点击即可下载包含当前显示论文信息的Markdown格式报告。

## 注意事项

- **后端服务**: 确保后端服务 (`app.py`) 已正确运行，否则前端无法获取数据。
- **网络连接**: `arxiv` 库需要网络连接来访问arXiv API。
- **查询逻辑**: 
    - 后端不再固定搜索 'agent' 主题，完全由用户输入决定。
    - 时间逻辑已修改为查找用户指定天数内的论文（从当天算起，例如7天即为包括今天在内的过去7天）。
- **报告生成**: 报告是基于前端当前获取和显示的论文数据生成的，通过POST请求将数据发送到后端进行处理。
